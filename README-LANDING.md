This folder contains the landing page components for the Myanmar Top Swiftlet project.

Setup notes:

- Add high-quality images to `public/images/` as listed in `public/images/readme.txt`.
- Run the dev server:

```bash
npm run dev
```

Design choices:

- Minimalist, premium palette with `#1F4D3A` primary and `#C9A227` accent.
- Tailwind CSS for utility styling and rapid layout.
- Framer Motion for subtle animation effects.
